            R.S. Mod RRRoamer • 2 years ago • edited

            I am pretty fond of the tag-connect cables, and have been using the 10-pin version for SWD headers consistently for some time now. It's a bit bigger than the 6-pin version of course, but still pretty comparable to the equivalent .127" pitch 10-pin male header. Plus, it has the major advantage of being free, which adds up if you're making more than a few designs. Those small headers are surprisingly expensive! I do recommend getting a few of the tag-connect clips though, so that you can keep the probe in place for extended debugging. This means you don't have to use the legged version of the connector, which does eat up rather a lot of space. I don't have any experience with the st-link in particular, but if it has the standard .127" swd header then it should work fine.
